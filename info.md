Package Name: sdg-term
Descriptive Name: SDG Terminal & Shell Configuration
Source: https://git.sdgcloud.nl/SDGDen/SDG-TERM
Maintainer: SDGDen <sdgden@sdgcloud.nl>
Version: 0.1.0

Dependencies: ghostty, zsh, oh-my-zsh, powerlevel10k, eza, zoxide, fzf, micro

Description: Terminal emulator + shell configuration module. Provides Ghostty terminal config (dankcolors theme, 50% opacity, 1GB scrollback), Zsh shell with oh-my-zsh, Powerlevel10k rainbow prompt, eza aliases, zoxide directory jumper, and cross-package integration (sdgfetch on startup, sdgtip random, SDG tool aliases). Includes dankcolors and noctalia themes.
